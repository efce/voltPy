Welcome Welcome to the Voltammetry Center, voltammetric signal processing and analysis web service!
Our goal is to create easy to use website, offering both novel/advanced and standard processing and analysis procedures.
Here you can upload, store, manage, search, process and analyze your voltammograms. The service is in rapid development, therefore, many new changes and features will arrive on regular basis and it is considered as an alpha version. However, there is wide selection of tools already available! We hope you'll enjoy during your time here. Please report any errors, bugs, feature requests etc. directly to our GitHub page.
This is an open source project, with the source code available at https://github.com/efce/voltPy/.
Glossary Short explanation of terms used in the web service:
  • File - file uploaded by the user and parsed into the database.
  • Fileset - collection of files uploaded together.
  • Dataset - basic block of the data used on the web service. Only datasets can be processed and analyzed. Dataset can be created out of whole file (create dataset in file view), whole fileset (create dataset in fileset view) or any selected curves via Create Dataset menu option.
  • Processing - processing is operation on the dataset which results in its change. Processing includes digital filtration and background corretion methods.
  • Analysis - analysis is the process which tries to obtain usefull information from the dataset. Analysis can be estamation of some measurement parameters or for example the standard addition method.
